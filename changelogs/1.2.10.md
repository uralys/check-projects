# v1.2.10 - 2025-12-22

## Features

### CLI Fetch Progress Bar

Added a visual progress bar when fetching remote repositories with the `-f` flag.

**What's New**:

- Real-time progress bar showing fetch status: `Fetching [████████░░] X/Y projects`
- Visual feedback during potentially long fetch operations

**Impact**: Better UX when fetching multiple repositories, users can now see progress instead of waiting blindly.
